# Einrichtung der Testumgebung – GroceryMate

## Testumgebung

### Hardware
- Laptop (Windows oder macOS)
- Internetverbindung

### Software
- Browser: Chrome, Firefox, Safari, Edge
- GitHub (für Dokumentation)
- Optional: PyCharm (für spätere Automatisierung)

### Testdaten
- Öffentliche Produktdaten der Website
- Keine Login-Daten erforderlich
- Keine Zahlungsdaten erforderlich

### Testsystem
- Live-System: https://grocerymate.masterschool.com/

### Hinweise
- Die Seite ist ein öffentliches Demo-System.
- Bestellungen können nicht abgeschlossen werden.
- Bewertungen sind nur nach einem Kauf möglich, daher nicht testbar.
- Warenkorb-Funktion ist eingeschränkt oder nicht verfügbar.
